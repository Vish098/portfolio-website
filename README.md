# Smart Study Planner

A simple and interactive web application to plan daily study schedules efficiently.

## 🚀 Features
- Add subjects with study hours
- Automatically calculates total study time
- Delete individual tasks
- Clear all tasks
- Displays warning for excessive workload

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🌐 Live Demo
https://vish098.github.io/smart-study-planner/

## 📌 Description
This project helps users organize their study time by adding subjects and tracking total hours. It uses JavaScript for dynamic updates and DOM manipulation.
